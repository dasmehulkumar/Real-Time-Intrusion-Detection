<h1>🔐 Intrusion Detection System</h1>

<h3>Real-Time Network Threat Detection using CICIDS2017 & Npcap</h3>

<p>
A machine learning-based Intrusion Detection System (IDS) that captures live network traffic and detects malicious activities in real time.
</p>

</div>

<hr>

<h2 align="center">📌 Project Overview</h2>

<p align="center">
This project focuses on building a real-time Intrusion Detection System (IDS) capable of analyzing network traffic 
and identifying cyber threats such as DDoS, brute-force attacks, and port scanning. 
It uses the CICIDS2017 dataset for training and Npcap for live packet capturing.
</p>

---

<h2 align="center">🎯 Objectives</h2>

<ul align="center" style="list-style-position: inside;">
<li>Monitor real-time network traffic</li>
<li>Detect malicious activities using ML models</li>
<li>Classify traffic as benign or attack</li>
<li>Bridge offline dataset learning with live traffic</li>
</ul>

---

<h2 align="center">⚙️ Key Features</h2>

<ul align="center" style="list-style-position: inside;">
<li>📡 Live Packet Capture (Npcap)</li>
<li>🧠 Machine Learning-based Detection</li>
<li>🚨 Real-time Intrusion Alerts</li>
<li>📊 Traffic Analysis & Feature Extraction</li>
<li>🔍 Detection of multiple attack types</li>
</ul>

---

<h2 align="center">🗂️ Dataset</h2>

<p align="center">
The CICIDS2017 dataset provides realistic labeled network traffic including both benign and malicious activities.
It includes attacks such as DDoS, brute force, botnet activity, and web-based attacks.
</p>

---

<h2 align="center">🌐 Npcap Integration</h2>

<p align="center">
Npcap is used to capture live network packets from the system. These packets are processed and converted 
into features, which are then analyzed by the trained machine learning model for real-time detection.
</p>

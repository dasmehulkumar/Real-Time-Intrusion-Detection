## 📌 Project Overview
This project focuses on building a real-time Intrusion Detection System (IDS) capable of analyzing network traffic and identifying cyber threats. It leverages the CICIDS2017 dataset for training and Npcap for live packet capturing, enabling real-time intrusion detection.

---

## 🎯 Objectives
- Monitor real-time network traffic  
- Detect malicious activities using machine learning  
- Classify traffic as benign or attack  
- Bridge offline dataset learning with live traffic  

---

## ⚙️ Key Features
- 📡 Live Packet Capture (Npcap)  
- 🧠 Machine Learning-based Detection  
- 🚨 Real-time Intrusion Alerts  
- 📊 Traffic Analysis & Feature Extraction  
- 🔍 Detection of multiple attack types  

---

## 🗂️ Dataset
The CICIDS2017 dataset provides realistic labeled network traffic including both benign and malicious activities. It includes attacks such as DDoS, brute force, botnet activity, and web-based attacks.

---

## 🌐 Npcap Integration
Npcap is used to capture live network packets from the system. These packets are processed into structured features and analyzed by the trained machine learning model for real-time detection.
